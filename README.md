# GulpPet · 咕嘟 🐾💧

> 懂你工作节奏的补水小伙伴

---

## 产品简介

GulpPet 是一款面向**长期对着电脑工作的知识工作者**的双端补水提醒应用。

它以**桌面灵宠**的形式常驻屏幕角落，感知你的工作状态，在你最可能停下来的时机发出低打扰提醒——不是弹窗轰炸，而是像一只懂你的小动物，在合适的时候戳你一下。

---

## 核心功能

| 功能 | 说明 |
|---|---|
| 🐾 桌面灵宠 | 透明浮窗，可拖拽，常驻桌面，不遮挡工作 |
| ⏱ 智能提醒 | 基于工作时段动态计算提醒间隔（30–90 min） |
| 🥤 一键记录 | 预设杯型快选，3 秒完成饮水记录 |
| 📊 今日进度 | 环形进度 + 时间线，一眼知晓今日状态 |
| 🌙 免打扰 | 全屏检测 + 自定义免打扰时段，开会不被打扰 |
| 📱 双端同步 | 桌面端（Electron）+ 手机端（React Native），Firebase 实时同步 |

---

## 技术方案

```
桌面端：Electron + React
手机端：React Native + Expo Go
数据库：Firebase Firestore（实时同步）
动画：  Rive（桌宠动效）
```
